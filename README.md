# 🦌 Agent Ralf

Extension Chrome pour automatiser le web en langage naturel.

Basé sur [PageAgent](https://github.com/alibaba/page-agent) par Alibaba, avec branding et personnalisation par Mak3it AI.

## 📥 Installation

1. Télécharger le ZIP depuis [mak3it.icu](http://mak3it.icu:8080/agent-ralf.zip)
2. Décompresser
3. Chrome → `chrome://extensions`
4. Mode développeur → ON
5. "Charger l'extension non empaquetée"
6. Sélectionner le dossier

## 🔧 Configuration

L'extension utilise une API compatible OpenAI. Configurez:
- **URL de base**: Votre serveur LLM (ex: Ollama Cloud, OpenAI, etc.)
- **Clé API**: Votre clé d'API
- **Modèle**: Le modèle à utiliser
